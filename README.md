# Ponto de Coleta Inteligente para Reciclagem

Este é um projeto web que visa promover a reciclagem e conscientização ambiental através de um sistema de pontos. Os usuários podem depositar materiais recicláveis em pontos de coleta inteligentes e receber pontos que podem ser trocados por prêmios.

## Link do Site Hospedado

acesse: https://ponto-de-coleta-inteligente-para-re.vercel.app

## Funcionalidades

- Sistema de pontos por materiais recicláveis
- Catálogo de itens válidos para reciclagem
- Ranking de impacto ambiental
- Informações sobre benefícios ambientais
- Prêmios disponíveis para troca de pontos
- Carrossel de imagens na página inicial
- Filtros na página de prêmios
- Formulário de contato
- Sistema de busca
- Seção de equipe atualizada

## Estrutura do Projeto

O projeto é composto por 5 páginas principais:

1. **index.html** - Página principal com informações sobre como funciona o sistema
2. **ganhe-premios.html** - Detalhes sobre os prêmios disponíveis e como trocar pontos
3. **itens-validos.html** - Lista de itens eletrônicos e componentes aceitos para reciclagem
4. **meio-ambiente.html** - Benefícios ambientais da reciclagem
5. **ranking.html** - Ranking de participantes com seus respectivos pontos

## Fotos da Equipe

As fotos da equipe foram adicionadas à pasta `fotos/` e estão sendo utilizadas:

- Na página de ranking para identificar os participantes
- Na página principal, a seção "Equipe Envolvida" mostra a gerente de projeto com sua foto

A foto de Camily Shultz aparece na seção "Equipe Envolvida" como gerente de projeto.
